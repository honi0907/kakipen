KakiMoni アセットフォルダ 説明書
================================

このフォルダに、KakiMoni で使う画像やテキストを置きます。
親機・子機は同じ assets フォルダを参照します。

フォルダ構成
------------

assets/
  README.txt           … この説明書
  seat-names.txt       … コンパネに表示する席名（1行目＝席1 … 10行目＝席10）
  backgrounds/         … 子機の書き画面・背景
  choices/             … 選択肢画像（全席に一斉表示）
  overlays/
    correct/           … 正解オーバーレイ
    incorrect/         … 不正解オーバーレイ
  logo/                … 子機の外部出力用ロゴ（GO 前の cover）

各サブフォルダには README.md もあります。詳細はそちらも参照してください。


backgrounds/（背景画像）
------------------------

・PNG / JPG / WEBP / GIF / TIFF が使えます
・16:9 の横長（例: 1920×1080）を推奨します

席 ID ごとに自動で背景を割り当てるには、次のようなファイル名にします。

  席1 … BG_ID1.png  または  BG_ID01.png
  席2 … BG_ID2.png  または  BG_ID02.png
  （席10 まで同様。大文字・小文字は区別しません）

対応ファイルが無い席は、背景なし（白）のままです。
画像は backgrounds/ フォルダの中に置いてください（assets 直下ではありません）。


choices/（選択肢画像）
----------------------

・基本は TIFF（アルファチャンネル付き）です
・PNG / JPG / WEBP / GIF も利用できます
・親機コンパネのドロップダウンで選び、「選択肢送信」で全子機に表示します
・フォルダに追加したら、設定ウィンドウの「アセット再読み込み」を押してください


overlays/（正解・不正解のオーバーレイ）
---------------------------------------

・親機コンパネで席を「正」「誤」にして「判定 GO」を押すと表示されます
・基本は TIFF（アルファチャンネル付き）です
・PNG / JPG / WEBP / GIF も利用できます
・ファイル名に fill を含めると、画面全体に合成表示されます
  （例: aka_fill.tiff / ao_fill.tiff）


logo/（外部出力ロゴ）
---------------------

・子機の外部モニター起動時、GO 前に全画面で表示する画像です
・PNG 推奨。複数ある場合はファイル名の昇順で先頭 1 枚が使われます
・無い場合は cover なしで書き画面を表示します


seat-names.txt（席名）
----------------------

・1行に 1 席分の名前を書きます（最大 10 行＝席1〜席10）
・親機の設定ウィンドウで「テキストネームを読み込む」を ON にすると反映されます


反映のしかた
------------

・画像を置き換えたあと … 親機の設定ウィンドウ →「アセット再読み込み」
・席ごとの背景 … ファイルを置けば、子機の次回接続から自動反映（親機の再起動は不要）
・seat-names.txt … 保存後、設定で ON にしていればコンパネに反映
