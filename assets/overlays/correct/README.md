# 正解オーバーレイ画像フォルダ

親機コンパネで席を「正」に設定して **判定GO** を押すと、子機タブレット・外部出力・コンパネプレビューに表示されます。

## 対応フォーマット

**TIFF（アルファチャンネル付き）を基本**とします。透過合成に向いています。

PNG / JPG / WEBP / GIF も利用できます。

## ファイル名に `fill` を含む場合

背景全体に合成表示されます（例: `aka_fill.tiff`）。

## デフォルト

画像未指定時は `/overlays/correct/aka_fill.tiff` が使われます（同ベース名の TIFF / PNG 等があればそちらを優先）。
