背景画像（PNG/JPG/WEBP/GIF/TIFF）をこのフォルダに配置してください。

## 席 ID ごとの自動背景

子機が接続すると、席 ID に対応するファイルを自動で書き画面の背景に使います。

| 席 ID | ファイル名の例 |
|------|----------------|
| 1 | `BG_ID1.png` または `BG_ID01.png` |
| 2 | `BG_ID2.png` または `BG_ID02.png` |
| … | … |

- 大文字・小文字は区別しません（`bg_id1.jpg` も可）
- 対応ファイルが無い席は背景なし（白）のままです
- 親機を再起動しなくても、ファイルを置けば次回接続から反映されます

## 推奨サイズ

16:9 の横長（例: 1920×1080）が書き画面に合います。
